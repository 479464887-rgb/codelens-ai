CodeLens AI - Privacy Policy

Last updated: June 3, 2026

1. What Data We Collect

CodeLens AI is designed with privacy in mind. We do not collect, store, or transmit any user data to our servers.

The extension operates entirely within your browser using local storage:

  • API Keys: Your DeepSeek, Kimi, and GitHub API keys are stored locally in your browser's chrome.storage.sync. They are never sent to any server other than the respective API endpoints.
  • Analysis History: A summary of your past analyses (PR title, repository name, timestamp) is stored locally in chrome.storage.local for your convenience.
  • Settings: Your preferences (language, auto-analyze toggle) are stored locally.

2. How Your Data Is Used

When you analyze a Pull Request:

  • The PR's code changes are fetched from GitHub's public API and sent directly to your chosen AI provider (DeepSeek or Kimi) for analysis.
  • Code data is transmitted only between your browser, GitHub, and the AI provider. We have no intermediate server — the extension makes direct API calls from your browser.
  • Analysis results are displayed in the extension panel and cached locally.

3. Third-Party Services

The extension may communicate with the following third-party services, solely for the purpose of providing its functionality:

  • GitHub API (Privacy Policy) — to fetch PR metadata and code diffs
  • DeepSeek API (Privacy Policy) — for AI-powered code analysis
  • Kimi/Moonshot API (Privacy Policy) — alternative AI provider
  • ExtensionPay (Privacy Policy) — for subscription management (only if you upgrade to Pro)

4. Data Retention

All data is stored locally in your browser. You can clear your analysis history at any time from the Settings page. Uninstalling the extension removes all local data.

5. Permissions

The extension requests the following Chrome permissions:

  • activeTab — to detect when you're on a GitHub PR page
  • storage — to save your settings and history locally
  • alarms — to reset daily usage counters
  • Host permissions: github.com, api.github.com, api.deepseek.com, api.moonshot.cn — to communicate with the respective APIs

6. Your Rights

Since no personal data is collected on our servers, there is no data to request, export, or delete through us. All data resides in your browser and is under your complete control.
